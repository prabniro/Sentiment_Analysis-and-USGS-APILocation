# Sentiment Analysis Project
Machine Learning Approach for Real-Time Disaster Response through Social Media Analysis
This study explore the use of the machine learning models and Natural Language Processing (NLP) techniques to transform the social media dataset from twitter (now X) to into actionable insights for emergency response in the events of disaster. CrisisNLP dataset from the 2015 Nepal Earthquake is utilised to study and addresses the critical challenge of information overload in disaster informatics.

The integration of USGS Api data supported the system by filtering out spatial noise from the distant location; shifting the focus on the disaster area.
